# isGSM7

> Check if a string conforms/validates to GSM-7bit.

## Installation

```bash
$ npm i --save isgsm7
```

## Usage

```js
const isGSM7 = require("isGSM7");

console.log(isGSM7("ðŸ”¥"));
// false

console.log(isGSM7("It was a teenage wedding and the old folks wished them well"));
// true
```
